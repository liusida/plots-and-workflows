# plots-and-workflows
A collection of plots with embedded ComfyUI workflows preserved in each PNG for easy reproduction and experimentation.

All plots are saved directly from ComfyUI, so the workflow and source code are embedded as PNG metadata. Simply drag any PNG into the ComfyUI interface to view the workflow, adjust parameters or code, and generate your own plots.

To access the embedded source code, install [ComfyUI](https://github.com/comfyanonymous/ComfyUI) (a node-based content-generation environment) along with my custom node [ComfyUI-Notebook](https://github.com/liusida/ComfyUI-Notebook) (a Jupyter-like notebook to run Python). You may need additional Python packages such as `matplotlib` if theyâ€™re not already installed.
Note: This does not work on ComfyUI Cloud because custom nodes cannot be installed there.
